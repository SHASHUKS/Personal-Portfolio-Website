# Personal Portfolio Website - Week 1 Project

## Project Overview
This project is a simple personal portfolio website created using HTML in Visual Studio Code. 
The purpose of this project is to learn the basics of HTML including web page structure, 
links, images, lists, and forms.

## Project Features
- Personal portfolio webpage
- Navigation menu with internal links
- About Me section with profile image
- Skills section with programming skills
- Contact section with email and form
- Simple and clean HTML structure

## Setup Instructions
1. Install Visual Studio Code
2. Create a project folder
3. Create a file named index.html
4. Add your profile image (profile.jpeg)
5. Open index.html in a web browser

## HTML Concepts Used
- HTML document structure
- Headings and paragraphs

- Lists (ul)
- Links and email links
- Images
